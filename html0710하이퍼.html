하이퍼링크가 필요한 이유 : HTML 문서 하나에 모든 정보를 담을 수 없기 때문에 여러 HTML 문서로 나누어 정보를 관리해야한다. 따라서 나누어진 HTML 문서를 특정한 문서규칙(IA)에 따라 서로 연결하여야 하는데 이때 필요한 것이 하이퍼링크이다.

절대경로 : HTML 문서의 위치와 상관없이 웹 페이지 또는 파일 위치를 나타내는 경로로 최상위 위치(root 디렉터리)에서 시작하는 파일의 경로를 말한다.

상대경로 : HTML 문서의 위치에 따라 결정되는 웹 페이지 또는 파일의 경로로 HTML 문서가 있는 디렉터리 위치가 중심이 되는 경로이다. 하위 디렉터리에 있는 문서의 경우 / 상위 디렉터리에 있는 문서의 경우 ../ 를 사용한다.

아이디(내부) 경로 : HTML 문서 내부에 있는 특정한 위치로 이동하고자 할 떄 사용한다.
