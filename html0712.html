웹 페이지는 공간 분할 태그를 사용하여 웹 페이지를 나눈다. 공간을 분할해야하는 이유? 공간을 분할해야 CSS로 원하는 레이아웃을 구성할 수 있기 때문이다. 공간을 분할하지 않으면 반응형 웹을 구현할 수 없다. 대표적인 공간 분할 태그 : div(블록형=width,height먹는데 개행),span(인라인형=width,height않되고,좌에서 우로 배치),img(인라인-블록형=width,heigth되고, 좌에서 우로 배치) 블록형 박스 : h1~h6,p,div 인라인형 박스 : a, i, input 인라인-블록형 박스 : img
